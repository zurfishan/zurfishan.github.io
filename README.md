## 👋 Hi, I'm Zurfishan

I'm a **Computer Engineering student** passionate about building things at the intersection of software and hardware. This blog is my digital garden — a place where I share what I learn, the projects I build, and the challenges I overcome.

I forked this Jekyll theme, `contrast`, to create my own corner of the web. You can find the original theme on [Niklas Buschmann's GitHub](https://github.com/niklasbuschmann/contrast).

---

## 🚀 About This Site

This website is built with **Jekyll** and hosted for free on **GitHub Pages**. It's designed to be simple, fast, and focused on content.

Here are some features you'll find:

- 🌙 **Dark mode** (automatic, based on your system preference)
- 📐 **KaTeX** support for beautiful math notation
- 🧩 **No external resources** — everything is self-contained
- 📁 **Optional sidebar** (not currently active)
- 📚 **Archive page** to browse all posts
- 🎨 **Syntax highlighting** for code snippets
- 💬 **Comment support** (via Disqus or Isso, if enabled)

## 🛠️ Running This Site Locally

Want to run this blog on your own computer? You'll need [Ruby](https://www.ruby-lang.org/en/documentation/installation/) installed. Then:

```bash
git clone https://github.com/zurfishan/zurfishan.github.io.git
cd zurfishan.github.io
gem install bundler jekyll jekyll-feed
bundle exec jekyll serve
